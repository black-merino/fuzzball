POI Sample
====

Apache POI を使った Excel ファイル読み込みのサンプルプログラム

### eclipse project について
C:\JavaClasses\poi-3.16 以下に poi 関連の jar を展開し、
外部jar として import しています。

異なるディレクトリに展開した場合は、
外部jar として import しなおしてご利用ください。
